Try it
---

Start server using docker:
```
$ make run
```

Example: 

http://127.0.0.1:8080/fill/400/100/www.audubon.org/sites/default/files/a1_1902_16_barred-owl_sandra_rothenberg_kk.jpg

Debug logs:

```
$ make logs
```


Tests
---

```
$ make test
```
